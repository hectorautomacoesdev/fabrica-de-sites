# Referência de API

Esta página é **gerada automaticamente** a partir dos *docstrings* do código-fonte, via
[mkdocstrings](https://mkdocstrings.github.io/). Quando o código e seus docstrings mudam, a
referência aqui acompanha — uma única fonte de verdade.

!!! info "Como funciona"
    O mkdocstrings usa o [griffe](https://mkdocstrings.github.io/griffe/) para analisar o
    pacote `fabrica_sites` (em `src/`) e renderizar assinaturas, tipos e docstrings.
